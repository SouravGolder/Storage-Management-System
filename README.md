# Storage-Management-System

## Tech Stack
- Node.js
- Express.js
- MongoDB

## Features
- Storage CRUD
- Item management
- Inventory stock in / stock out
- Modular architecture

## Setup
```bash```
git clone <repo-url>
npm install
npm run dev


````API Endpoints````

POST /api/storages

GET /api/storages

PUT /api/storages/:id

DELETE /api/storages/:id

POST /api/items

POST /api/items/:id/stock-in

POST /api/items/:id/stock-out
